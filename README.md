### 第三章

第三张主要是学习元件的概念。

元件(symbol)可以用于特效、东校或交互性可重用的资源。有三种：图形、按钮和影片剪辑。可以把元件视作原始的摄影底片，而把“舞台”上的实例视作底片的相片，只需利用一张底片，即可创建多种相片。

![练习图片](http://i13.tietuku.com/1c34bdf40f4fec7f.png)
